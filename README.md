# hshop_terminal

<br>For Windows:</br>
    <br>    + Install JRE 1.8.0 or above.</br>
    <br>    + Launch app via Hshop Terminal.exe.</br>
<br></br>    
<br>For Linux:</br>
    <br>+ Install JRE 1.8.0 or above</br>
    <br>+ Launch app via cmd: "java -jar Hshop Terminal.jar" or create a launcher with that cmd.</br>
